# scopy — 채용 시장 탐색 대시보드 (v2.4)

원티드(Wanted) 공개 API(V2 `/jobs` + V1 검색·기업·인사이트·태그) 기반의 **구직자용**
채용 시장 대시보드. 기획 문서는 [prd.md](prd.md) 참고.

> v1(기업 채용담당자용 ATS 대시보드)은 방향 변경으로 폐기·재구축함.
> v2.4부터 시장 개요·공고 탐색·기업 비교·북마크·추천은 모두 원티드 실 API 데이터를 쓴다. 권한이 없는 기업 인사이트 지표는 가상 값으로 채우지 않고 화면에서 제외한다.

## 실행

```bash
open index.html        # 브라우저에서 바로 열면 됨 (빌드 불필요)
```

## 화면

1. **시장 개요** — 진행중 공고(직군별 전수 집계)·채용 기업·보상금·신입 KPI + 직군/스킬/경력대/지역/매력태그 차트 + 마감 임박 공고 (실 API)
2. **공고 탐색** — 검색·직군·경력·정렬 필터 + 공고 카드 (D-day, 북마크), 카드 클릭 시 실제 지원 페이지로 이동 (실 API)
3. **기업 비교** — 실 공고 표본을 기업 단위로 집계한 비교표 + 기업 2곳 A/B 레이더 비교
4. **북마크** — 저장한 공고 (localStorage, 실 API)
5. **마이페이지** — 자격증 검색·축약/전체보기, 외부활동, 선호 직군 1~3순위, 직군별 준비도, TXT 자소서 파일 관리·추천 기업, 항목 선택형 자동 이력서 미리보기·DOCX 다운로드

## 데이터 재생성

```bash
cd db
./export_data.sh        # 자격증 카탈로그 재생성 + 원티드 실 API 재호출
```

`export_data.sh`는 내부적으로 `fetch_live_jobs.py`를 호출해 원티드 실 API를 다시 불러온다.
또한 공식 원본 CSV와 지표 보강 CSV를 병합해 `js/certifications.js`로 변환한다. 정적 파일을
직접 여는 구조라 브라우저에서 CSV를 직접 요청하지 않고, CSV를 원본으로 유지한 채 JS로 내보낸다.
`.env`(git 제외)에 `WANTED_CLIENT_ID`/`WANTED_CLIENT_SECRET`이 없거나 API 호출이 실패하면
직전 `js/data.js`의 `liveJobs`를 유지하며, 이전 데이터도 없을 때만 빈 배열로 진행한다.

## 자격증 데이터

- 기본 목록: 한국산업인력공단 국가기술자격 종목별 시험정보 488종과 국가·공인민간자격 보강 항목
- 시험 인기: 2025 국가기술자격통계연보의 2024년 접수 건수
- 기업 우대: 한국산업인력공단 2021년 수험자 조사에서 공개된 종목별 비율

숫자가 확인되지 않은 자격에는 임의 점수를 넣지 않는다. 카탈로그에 없는 자격증·수상도
직접 입력해 저장할 수 있으며, 공식 목록에서 선택한 경우에만 유형과 통계 배지가 표시된다.

## 마이페이지 프로필·이력서

- 선호 직군은 1~3순위까지 저장하며 1순위를 공고 탐색의 기본 필터로 사용한다.
- 선호 직무는 마이페이지 안내 문구와 같은 가로축에 두고, 아래 외부활동 제목의 왼쪽 세로선에 시작점을 맞춘 타원형 버튼 3개에서 확인·선택한다. 타원은 직무명 길이만큼 오른쪽으로 확장된다.
- 보유 자격증은 처음 3개만 노출하고, 4개부터는 `외 N개` 전체보기 팝업에서 확인한다.
- 자격증은 취득일을, 외부활동은 활동일을 날짜 선택기로 지정해 함께 저장한다.
- 외부활동은 프로젝트·인턴·공모전·대외활동·교육·봉사로 구분해 저장한다.
- 외부활동도 처음 3개만 노출하고, 4개부터는 `외 N개` 전체보기 팝업에서 확인한다.
- 직군별 준비도는 공개 합격자 원본 데이터가 아닌 대표 자격·활동 벤치마크 체크리스트다.
  실제 합격확률이나 합격 보장을 의미하지 않는다.
- 준비도 카드에는 직군별 프로젝트·공모전·인턴·대외활동의 구체적인 주제와 포트폴리오에 남길 결과물 예시를 함께 표시한다.
- 자소서는 TXT 파일로 등록해 브라우저에서 관리하며, 읽어온 본문을 회사·공고 추천과 자동 이력서에 반영한다.
- 자동 이력서는 직무별 양식과 포함할 자격증·외부활동을 선택해 미리보기를 만들고 `.docx` 파일로 다운로드한다.
- 공고 탐색의 AI 맞춤 회사·공고 추천은 별도 생성형 AI 호출 없이 선호 직군, 자격증, 대외활동, 자소서 본문과 실제 공고를 비교하는 설명 가능한 로컬 추천이다.

## API 활용 확장 여지

지금 안 쓰는 엔드포인트 중 이 앱 범위를 벗어나지 않고 더 붙일 수 있는 것들은 prd.md §3-1 참고
(검색 자동완성, 기업의 다른 공고 보기, 스킬 사전 동기화). `/insight/company`(권한 없음)와
`/ai/*`(유료 계약 전용)는 이 파트너 키로는 못 붙는다.
